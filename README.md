✅ Full GDPR Data Processing Addendum - Compliant with EU, UK, and worldwide data protection laws
✅ 7-Day Kill-Switch - Either party can terminate with 7 days notice
✅ Late Payment Interest - Default 1.5% monthly (editable)
✅ Dispute Venue - Set to your country/city (editable fields)
